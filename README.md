# K230视觉采集与智能处理系统

## 基本考核项：  
（1）熟练使用Python编程语言在嵌入式AI开发板上进行编程和操作；  
（2）熟练使用嵌入式开发板上的摄像头采集图像数据，原始数据类型不少于4种，其中人体数据2种以上，物体数据2种以上；  
（3）采集的原始数据存储在嵌入式AI开发板内置的存储器中；  
（4）测试时，直接调用开发板上的显示屏显示采集和存储的图像数据。  
## 拓展考核：  
（5）可以通过联网方式，将采集数据上传到PC机或者手机上面进行显示；  
（6）所有操作基于Github开源平台进行，熟悉团队代码管理方法和团队协作；  
（7）通过特征提取的方式，将简化数据进行存储；  
（8）其他的个性化操作。
